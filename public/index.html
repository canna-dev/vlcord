VLCord Setup Instructions

VLC Setup

Enable VLC's HTTP interface by adding these parameters when launching VLC:

--intf http --http-host localhost --http-port 8080 --http-password vlcpassword

In VLC, you can also go to:

  1. Tools → Preferences
  2. Show all settings (bottom left)
  3. Interface → Main interfaces → Check "Web"
  4. Interface → Main interfaces → Lua → Lua HTTP → Set your password

Discord Application Setup

  1. Go to Discord Developer Portal
  2. Create a new application
  3. Copy the Application ID to the Client ID field in VLCord settings
  4. Under Rich Presence → Art Assets, upload these images:
    • vlc - VLC logo
    • vlcord - VLCord logo
    • play - Play icon
    • pause - Pause icon

TMDb Setup

  1. Sign up at TheMovieDB
  2. Go to Settings → API → Create new API key
  3. Copy the API Key (v3 auth) to the TMDb API Key field in VLCord settings
